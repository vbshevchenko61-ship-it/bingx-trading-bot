Основные файлы бота:

telegram_bot.py — основной Telegram-бот, который отслеживает BingX и отправляет сделки
app.py — Flask-обёртка для совместимости с хостингом
main.py — точка входа для развёртывания через Gunicorn

Конфигурация:
pyproject.toml — конфигурация зависимостей Python
replit.md — документация проекта и настройки

Веб-дашборд (опционально):
templates/base.html — базовый шаблон веб-дашборда
templates/index.html — интерфейс дашборда
static/style.css — стилизация дашборда
bot_service.py — альтернативный класс сервиса бота (в данный момент не используется)
